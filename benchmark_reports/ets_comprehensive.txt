=== Key Findings Summary ===
1. Exfoil uses direct function calls (compile-time optimized)
2. ETS uses runtime hash table lookups
3. Exfoil should be faster for frequently accessed static data
4. ETS is better for dynamic data that changes frequently
5. Memory usage: Exfoil stores data in module bytecode
6. Memory usage: ETS stores data in separate memory space
